# ARGOS-NT Case Input Template v2

Preencha os campos abaixo para criar um novo caso manualmente. Os campos obrigatórios estão marcados com *.

---

case_name: <nome do caso>*
email: <email principal>*
username: <username> (opcional)
target_name: <nome completo ou parcial>*
telefone: <telefone> (opcional)
location: <localização> (opcional)
birthdate: <data de nascimento> (opcional)

# Observações
- Se o nome do alvo for parcial, os resultados do scanner podem ser inconsistentes.
- Campos opcionais ajudam a enriquecer as buscas e dorks, mas não são obrigatórios.

---
Exemplo:

case_name: acme_investigation
email: john.doe@example.com
username: johndoe

# Nome parcial
# target_name: John
# Nome completo
# target_name: John Doe

target_name: John Doe
telefone: +5511999999999
location: São Paulo, Brazil
birthdate: 1985-07-21
